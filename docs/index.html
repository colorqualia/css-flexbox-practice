Flexible Box Layout Module

align-items

縦方向の配置
flex-start / center / flex-end / baseline / stretch
初期値はstretch

flexbox vertical top

.flex-vertical-top {
   display: flex;
   -webkit-align-items: flex-start;
           align-items: flex-start;
}

flexbox vertical middle

.flex-vertical-middle {
   display: flex;
   -webkit-align-items: center;
           align-items: center;
}

flexbox vertical bottom

.flex-vertical-bottom {
   display: flex;
   -webkit-align-items: flex-end;
           align-items: flex-end;
}

justify-content

横方向の配置
flex-start / center / flex-end / space-between / space-around
初期値はflex-start

flexbox horizontal left

.flex-horizontal-left {
   display: flex;
   -webkit-justify-content: flex-start;
           justify-content: flex-start;
}

flexbox horizontal center

.flex-horizontal-center {
   display: flex;
   -webkit-justify-content: center;
           justify-content: center;
}

flexbox horizontal right

.flex-horizontal-right {
   display: flex;
   -webkit-justify-content: flex-end;
           justify-content: flex-end;
}

flexbox horizontal space between

space-between両端揃え(フレックスアイテムの間に余白挿入)

.flex-horizontal-space-between {
   display: flex;
   -webkit-justify-content: space-between;
           justify-content: space-between;
}

flexbox horizontal space around

space-around両端揃え(フレックスアイテムの左右に余白挿入)

.flex-horizontal-space-around {
   display: flex;
   -webkit-justify-content: space-around;
           justify-content: space-around;
}

flexbox vertical horizontal center

.flex-vertical-horizontal-center {
   display: flex;
   -webkit-align-items: center;
           align-items: center;
   -webkit-justify-content: center;
           justify-content: center;
}
